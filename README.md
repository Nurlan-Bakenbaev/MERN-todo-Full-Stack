# ToDo App with MERN Stack

This ToDo app is built using the MERN stack (MongoDB, Express, React, Node.js) and features CRUD operations for managing your tasks. It also includes Material-UI components for a sleek user interface and Tailwind CSS classes for styling. Additionally, it has simple authentication functionality.

## Features

- Create, Read, Update, and Delete (CRUD) operations for managing tasks.
- Tasks can be marked as completed or important, with corresponding style changes.
- User authentication to keep your tasks private.


- **Frontend:**
  - React
  - Material-UI for UI components
  - Tailwind CSS for styling

- **Backend:**
  - Node.js
  - Express.js for the server
  - MongoDB for data storage
  - Mongoose for MongoDB object modeling
